類似機種の作業手順の展開

[作業手順]シートに製品の品番を入力すると、型番に応じた作業手順書へ自動的に差し替えられます。

品番の指定方法

品番には、ワイルドカード「*」を使用できます。ワイルドカードを使用すると、特定のパターンに一致する複数の品番を指定できます。例えば、 品番に「ITEM-100-RED」「ITEM-100-BLUE」「ITEM-100-YELLOW」の3種類がある場合、「ITEM-100-*」と入力することで、それらすべてを指定することが可能です。

品番によってタクトタイムが異なる場合

[品番]と[工程#]、[タクトタイム]を入力してください。[工程#]には、工程シートの名前に付けた連番を入力してください。

品番によって図解が異なる場合

[品番]と[工程#]、[作業手順#]、[画像]を入力してください。

  • [作業手順#]には、作業手順の連番を入力してください。
  • [画像]には、指定する画像のファイル名を入力してください。
  • 画像ファイルは、Excelファイルと同じフォルダにある「images」フォルダに保存してください。

品番によってチェック項目が異なる場合

[品番]と[工程#]、[作業手順#]、[チェック項目#]、[項目名]を入力してください。また、チェック項目を表示しない場合は、[表示]を「非表示」に変更してください。

  • [チェック項目#]には、チェック項目の連番を入力してください。
  • 必要に応じて、[基準値(最小)]と[基準値(最大)]、[初期値]を設定してください。
  • チェック項目を非表示にする場合は、[表示] を「非表示」に変更してください。
